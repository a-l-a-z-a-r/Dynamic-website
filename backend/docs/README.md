# API Documentation

## OpenAPI / Swagger

- Swagger UI: `http://localhost:5000/api/docs`
- OpenAPI JSON: `http://localhost:5000/api/docs-json`

The Swagger UI is generated from the NestJS application at runtime.

## AsyncAPI (RabbitMQ Events)

- AsyncAPI spec: `backend/docs/asyncapi.yaml`

You can visualize this file in the AsyncAPI Studio:
https://studio.asyncapi.com/
